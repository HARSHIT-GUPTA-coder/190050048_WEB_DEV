Projects

Reinforcement Learning Based Stock Trader

Reinforcement Learning Based Stock Trader

This project is done under the Institute Technical Summer project of IIT Bombay. The idea is to investigate various algorithms of Reinforcement Learning amd apply them to the domain of Stock Trading. We applied the DQN, DDQN, DDPG and DRQN algorithms to maximize the profit that we get from trading.

Face Recognition Using PCA

Face Recognition Using PCA

This project is done under the Summer of Code of IIT Bombay. The project involved learning a very useful and powerful algorithm called Principal Component Analysis and then finally implementing it and using it for the task of face recognition.

Github Repo

Visual Sudoku Solver

Visual Sudoku Solver

This is a project that will use OpenCV in Python to recognise the sudoku grid and digits which will be classified using a neural network and then a C++ utility will solve the sudoku and output the final solved state. This project is still on-going although majority of work is done.

Github Repo

Line-follower-on-Gazebo

Line-follower-on-Gazebo

This project aims at simulating a line follower bot using gazebo. There are two different models- the track and the bot. The bot uses a camera to get the image of path below it and then detect the path using OpenCV. A very basic feedback mechanism is implemented to control the bot.
